# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-19

### Added

- Initial release of the bookmarklet tool
- DuckDuckGo search engine redirect support
- Parameter extraction from Google search URLs
- Advanced UI modal for parameter selection
- Support for preserving various Google search parameters:
  - Search query (always preserved)
  - Time-based filters (past day, week, month, year)
  - Language restrictions
  - Country restrictions
  - Number of results
  - Safe search settings
  - Search type (images, videos, news, etc.)
- Build system for minifying and bundling bookmarklets
- Test page for local development and testing
- Pre-commit hooks for code quality (Prettier, ESLint)
- Comprehensive documentation (README, CHANGELOG)

### Features

- **Easy Installation**: Drag-and-drop bookmarklet installation
- **User-Friendly Interface**: Clean modal with checkboxes for parameter selection
- **Flexible Options**: Choose which parameters to preserve or search query only
- **Privacy-Focused**: Redirect to DuckDuckGo for privacy-enhanced searching
- **Development Tools**: Test page and build system for easy development
